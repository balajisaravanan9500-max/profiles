# Profile (Static HTML)

This is a simple static profile page. Quick run and development instructions:

## Run (Python HTTP server)

From the project root (`C:\Users\Balaji\Desktop\profile`) run:

```powershell
# Start server on port 8000
py -m http.server 8000
```

Open http://localhost:8000/ in your browser. The served file is `html`.

To stop the server press `Ctrl+C` in the terminal where it is running.
